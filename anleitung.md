# Megakino Mega Downloader - Ausführliche Anleitung

Willkommen beim Megakino Mega Downloader! Hier erfährst du alles, was du wissen musst, um das Programm wie ein Profi zu nutzen.

---

## 📦 1. Installation

Du kannst das Programm auf zwei Arten installieren:

### Methode A: Der schnellste Weg (über PIP)
Perfekt für alle, die sofort loslegen wollen.
1. Öffne dein Terminal (Windows: CMD oder PowerShell, Mac/Linux: Terminal).
2. Tippe: `pip install megakino-mega-downloader`
   *(Hinweis: Falls du einen Mac hast, nutze lieber `pip3`)*.

### Methode B: Für Entwickler (Lokal)
Wenn du den Code heruntergeladen hast und ihn in einer sauberen Umgebung (venv) nutzen willst:
1. Gehe im Terminal in den Programm-Ordner.
2. Erstelle eine Umgebung: `python3 -m venv venv`.
3. Aktiviere sie:
   - Windows: `.\venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. Installiere alles: `pip install .`

---

## 🚀 2. Das Programm nutzen

Tippe einfach `megakino` in dein Terminal. Du siehst dann dieses Menü:

*   **Search Movie/Series:** Gib einen Namen ein (z.B. "One Piece"). Das Programm sucht sofort auf Megakino und VOE nach passenden Inhalten.
*   **Settings:** Hier kannst du einstellen, in welchen Ordner deine Filme gespeichert werden sollen (Standard: Downloads).
*   **Deinstallations-Hilfe:** Falls du das Programm irgendwann mal löschen willst, zeigt dir dieser Punkt genau, wie du alles restlos entfernst.

### Tipps zur Bedienung:
- Bei der **Episoden-Auswahl**: Drücke die **Leertaste**, um eine Folge zu markieren (ein gefüllter Kreis erscheint). Drücke danach **Enter** zum Bestätigen.
- Mit **Strg + A** kannst du alle Folgen einer Serie auf einmal auswählen.

---

## 🛠️ 3. Wichtige Werkzeuge (Zusatzprogramme)

Damit alles perfekt läuft, solltest du zwei kostenlose Programme auf deinem Rechner haben:
1. **mpv:** Das ist der Video-Player. Ohne ihn kannst du die Filme nicht direkt im Programm "anschauen".
2. **FFmpeg:** Das ist das wichtigste Werkzeug im Hintergrund. Es sorgt dafür, dass die heruntergeladenen Filme sauber verarbeitet werden, damit sie auch auf deinem **iPhone oder Mac** abspielbar sind.

**Keine Sorge:** Wenn du das Programm startest und eins davon fehlt, zeigt dir Megakino Mega Downloader direkt den passenden Befehl an, um es mit einem Klick zu installieren!
