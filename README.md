## Deutsch <a name="deutsch"></a>

### 🌟 Was ist der Megakino Mega Downloader?
Willst du Filme und Serien in bester Qualität genießen, ohne dich mit nerviger Werbung oder komplizierten Webseiten rumzuschlagen? Dann ist dieses Tool genau das Richtige für dich! Der **Megakino Mega Downloader** ist ein blitzschnelles, modernes Terminal-Programm, mit dem du deine Lieblingsinhalte von Megakino und VOE suchen, direkt anschauen oder für unterwegs als MP4 speichern kannst.

### 🚀 Die Mega-Features:
*   **Blitzschnelle Suche:** Findet sofort, was du suchst.
*   **Parallele Downloads:** Lade mehrere Folgen gleichzeitig mit Höchstgeschwindigkeit herunter.
*   **Apple-Ready:** Alle Downloads werden automatisch als saubere, hochwertige `.mp4` gespeichert, die perfekt auf iPhone, iPad und Mac laufen.
*   **Intelligenter Fallback:** Wenn ein Link bei Megakino nicht geht, sucht das Programm automatisch bei VOE weiter.
*   **Zusammen schauen (Syncplay):** Schau Filme synchron mit deinen Freunden über das Internet!
*   **Auto-Update:** Das Programm sagt dir sofort Bescheid, wenn es eine neue Version gibt.

### 📦 Installation (Der schnellste Weg)

Es ist super einfach! Du brauchst nur Python auf deinem Rechner.

#### 1. Direkt über PIP (Empfohlen)
Öffne dein Terminal (oder PowerShell) und gib ein:
```bash
pip install megakino-mega-downloader
```
*(Falls `pip` nicht geht, probiere `pip3` oder `python -m pip install megakino-mega-downloader`)*

#### 2. Starten
Tippe einfach:
```bash
megakino
```
...und schon geht's los! Alternativ kannst du auch `mega-downloader` tippen.

### 🛠️ Voraussetzungen
Für das beste Erlebnis empfehle ich:
*   **mpv:** Damit du die Filme direkt im Programm starten kannst.
*   **FFmpeg:** Damit die MP4-Dateien perfekt für Apple-Geräte zusammengefügt werden.
*(Falls etwas fehlt, zeigt dir das Programm beim Start genau an, wie du es installierst!)*

## English <a name="english"></a>

### 🌟 What is Megakino Mega Downloader?
Tired of slow websites and annoying ads? Get your favorite movies and series directly to your PC! The **Megakino Mega Downloader** is a professional, high-speed CLI tool designed to search, watch, and download content from Megakino and VOE with ease.

### 🚀 Key Features:
*   **Concurrent Downloads:** Download multiple episodes at once with beautiful progress bars.
*   **Apple Compatibility:** Automatically saves files as high-quality `.mp4`, fully compatible with iPhone, iPad, and Mac.
*   **Smart Fallback:** If a stream is down on one provider, it automatically finds a working one on the other.
*   **Syncplay Support:** Watch movies perfectly in sync with your friends online.
*   **Modern UI:** A beautiful interactive terminal interface.
*   **Auto-Update Check:** Never miss an improvement!

### 📦 Quick Installation

#### 1. Via PIP (Easiest way)
Open your terminal and run:
```bash
pip install megakino-mega-downloader
```
*(If `pip` is not recognized, try `pip3` or `python -m pip install megakino-mega-downloader`)*

#### 2. Usage
Just type:
```bash
megakino
```
...or use the alias `mega-downloader`.

### 🛠️ Prerequisites
For full functionality, we recommend:
*   **mpv:** For direct video playback.
*   **FFmpeg:** For perfect MP4 container remuxing (Apple compatibility).
*(The app will show you exact install commands if any of these are missing!)*
